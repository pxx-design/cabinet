# Notice — reference images

Cabinet is a tool for *describing* visual technique. To make the seed library
usable out of the box, this repository ships a small set of **low-resolution**
reference images. They are included for study, commentary, and identification —
not as original work of this project.

## What's included

- **`assets/` + `thumbs/`** — 6 curated reference specimens (the seed library).
  Each is downscaled to a low resolution. The facet text *about* each image
  (the move descriptions in `data-v3.js`) is original to this project and is
  MIT-licensed; the images themselves are not.
- **`folio/` + `shots/`** — low-resolution thumbnails and short preview clips
  used by the "find inspiration" tab as a **link directory** pointing to
  third-party design sites and portfolios (Awwwards, CSSDA, Godly, Mobbin,
  Lapa, Refero, and individual studios/portfolios). They function as
  "visit this site" cards.

## Provenance

These references were collected from the public web over time and individual
source URLs were not recorded per image, so this notice cannot list a link for
every file. All rights in the images and clips remain with their respective
owners and original authors. No endorsement is implied.

## Takedown

If you own rights to any image here and want it removed or attributed
differently, open an issue on this repository (or contact the maintainer) and
it will be taken down promptly. We default to removal on request.

## For your own library

When you ingest your **own** references via the `/ingest` flow, you control
what enters `assets/` — keep this in mind before redistributing a fork that
bundles third-party images.
